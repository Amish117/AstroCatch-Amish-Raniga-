AstroCatch is an arcade-inspired video game in which players control an astronaut and score points by "catching" supply crates as they fall while missing meteors falling at random speeds. The game increases in difficulty by increasing meteor speeds, scaling as the player's score increases as well. The prototype game was built using Microsoft MakeCode Arcade using Block-based coding and Python in order to create a better understanding of event-based programming, sprite-based programs, variables, and conditional logic. The intent of the project is to create a retro style arcade game and to build a foundational code literacy perspective through game development, through game mechanics and player experience.
